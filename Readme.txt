				Calc v1.0

Программа вычисляет результат арифметических выражений, введенных в консоли.
Для вывода результата, после введения выражения, нужно ввести знак '=' , и нажать 'Enter'.
Для работы с десятичными дробями можно пользоваться знаками '.' или ','
Поддерживается следующие операции: '+' , '‐' , '*' , '/' , '^' (возведение в степень).
Приоритет операций: '^' ‐ высший , '*' , '/' ‐ средний, '+' , '‐' ‐ низший.
Поддерживается использование скобок, с произвольной степенью вложенности.
Под выражением "25(1+1)(‐2)(3)5" подразумевается произведение из четырех множителей (знаки '*' можно не ставить).
Между числами и знаками операций можно использовать пробелы.
Наберите "\Exit", "\exit" или нажмите Ctrl+z и нажмите Enter для выхода из программы.
Наберите "\help" в окне программы для вывода справки.

		Файлы:

"Calc (file version).exe" - 	запуск файловой версии программы. Эта версия вместо ввода данных с консоли читает
				файл "BIN\input.txt", а результат записывает в файл "BIN\output.txt".
"Calc.exe" - 			запуск стандартной версии программы. Работает с консолью. Данные вводятся 
				пользователем  в консоль при помощи клавиатуры и выводятся также в консоль.
"Input file Edit.exe" - 	запуск notepad'а для редактирования файла "BIN\input.txt".

"BIN\Calc (file).exe" - 	файловая версия программы (см. выше).
"BIN\Calc.exe" - 		стандартная версия программы.
"BIN\input.txt" -		входной файл (источник данных) файловой версии программы (см. выше).
"BIN\output.txt" - 		выходной файл (вывод результата) файловой версии программы (см. выше).
"BIN\input(check).txt" - 	готовый входной файл для проверки работы программы. Имеет при раздела:

				Errors -	в данном разделе все строки должны выдвть ошибку
				No errors - 	в данном разделе ошибок быть не должно
				Calculate - 	вычисление выражений и сравнение с ответами, должно совпадать
